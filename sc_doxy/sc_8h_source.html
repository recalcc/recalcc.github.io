00108 /*
00109  * Some not too obvious things about the flags:
00110  *    is_valid means there is a valid number in v.
00111  *    is_locked means that the cell cannot be edited.
00112  *    is_label set means it points to a valid constant string.
00113  *    is_strexpr set means expr yields a string expression.
00114  *    If is_strexpr is not set, and expr points to an expression tree, the
00115  *        expression yields a numeric expression.
00116  *    So, either v or label can be set to a constant. 
00117  *        Either (but not both at the same time) can be set from an expression.
00118  */
